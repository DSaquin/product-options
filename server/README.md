### CRUD API Endpoints

---

`GET` /products/random

`GET` /products/:itemId

`POST` /products/:itemId

`PUT` /products/:itemId

`DELETE` /products/:itemId

#### Parameters

| Name   | Type   | Description       |
| ------ | ------ | ----------------- |
| itemID | number | Id of the product |
| random | string | random product Id |
